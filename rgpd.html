Politique de Confidentialité

Collecte des Données

Nous collectons les informations suivantes :

  • Nom de code (pseudonyme).
  • Mot de passe (hashé et sécurisé).
  • Messages échangés via le chat (chiffrés).

Utilisation des Données

Les données sont utilisées pour :

  • Authentifier les utilisateurs.
  • Permettre la communication sécurisée entre agents.

Protection des Données

Nous utilisons des mesures de sécurité avancées pour protéger vos données :

  • Chiffrement des messages avec le code César.
  • Hachage des mots de passe.
  • Accès restreint à la base de données.

Droits des Utilisateurs

Conformément au RGPD, vous avez le droit de :

  • Accéder à vos données.
  • Modifier ou supprimer vos données.
  • Demander la suppression de votre compte.

Contact

Pour toute question ou demande, contactez-nous à : contact@shadowcomm.com.
